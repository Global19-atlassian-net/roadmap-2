Masa Depan Cerah dari io.js

Sebuah Roadmap Berbasis Komunitas

Untuk komunitas, dari komunitas

io.js adalah sebuah pengganti drop-in untuk node.js dan cocok dengan hampir semua modul di npm.

io.js adalah sebuah proyek open source berbasis komunitas. Ini dimulai oleh kontributor-kontributor utama node.js tetapi dalam beberapa bulan-bulan awalnya menarik lebih banyak developer-developer aktif daripada proyek node.js punya sepanjang sejarahnya.

Apa yang orang-orang mau dari io.js?

  1. Stabilitas
  2. Transparansi (Debugging dan Tracing)
  3. Streams yang Lebih Baik
  4. Dukungan Jangka Panjang
  5. Merangkul Standar-standar yang Sedang Berjalan
  6. Lokalisasi

Kebijaksanaan Stabilitas

Kami punya ekosistem package terbesar dari semua bahasa, kami tidak bisa merusaknya.

  • Kami tidak menghapus API JS.
  • Semantic Versioning
    • Penambahan API apapun berarti sebuah penaikan versi minor.
    • Perubahan di API C++ V8 ditangani oleh nan berarti sebuah penaikan versi minor dan apabila mereka tidak bisa berarti sebuah penaikan versi mayor.

Dari bagian Kebijaksanaan Stabilitas dari Roadmap. Kontributor dipersilakan masuk!

NG (Next Generation)

  • Sebuah platform baru untuk Aplikasi-aplikasi JavaScript modern.
  • Akan tetap backward compatible tanpa patokan waktu.
  • API baru dapat berpusat pada ES6/7 tanpa merusak modul-modul dan API yang lama.
  • Memisahkan kerja dalam API untuk kedepannya dari penyempurnaan untuk platform terkini.

Diskusi NG selalu berlangsung dalam repo NG.

Debugging & Tracing

Core selalu mengizinkan sebuah ekosistem alat-alat debugging dan tracing yang sangat besar dan beraneka ragam daripada mengesahkan high level abstractions seperti domains.

  • Berintegrasi dengan trace_event (API trace baru untuk V8 & Chrome).
  • Dokumentasi dan meng-evangelize sebuah ekosistem debugging.
  • Menyatukan system tracing endpoint (DTrace, LTTng, etc)

Pekerjaan dipimpin oleh Tracing Working Group. Kontributor dipersilakan masuk!

Streams

  • Streams Working Group telah mengambil alih readable-stream, yang adalah modul resmi io.js stream.
  • Memperbaiki semua masalah-masalah kompatibilitas yang ada.
  • Menyederhanakan pembuatan stream untuk menghindari error pengguna.
  • Menjelajah interface WHATWG Streams dan mengidentifikasi masalah-masalah kompatibilitas.
  • Meningkatkan performa stream.

Pekerjaan dipimpin oleh Streams Working Group. Kontributor dipersilakan masuk!

Dukungan Jangka Panjang

  • Jalur rilis baru akan diluncurkan dengan ketergantungan stabil dan yang didukung.
  • Rilis-rilis patch dari versi yang lebih tua akan tetap dipublikasikan selama orang-orang aktif berkontribusi.
  • io.js akan bertanggung jawab untuk versi tua dari ketergantungan-ketergantungan seperti V8 yang tampil di rilis-rilis terdahulu ketika pemeliharanya berhenti mendukung mereka.

Ringkasan dari Kebijaksanaan Stabilitas. Kontributor dipersilakan masuk!

Lokalisasi

  • Mengurangi dampak dari pemuatan icu (binary dan memory footprint).
  • Memelihara sebuah komunitas lokalisasi yang giat.
    • Komunitas bahasa yang swatantra (30+ telah mendaftar sejauh ini)
    • Usaha evangelism untuk memelihara keanggotaan (160+ anggota sejauh ini)
    • Mengembangkan tooling untuk menerjemahkan website, dokumentasi API, installer dan slide-slide presentasi.

Anda dapat terlibat dengan bergabung dengan komunitas untuk bahasa anda :)
